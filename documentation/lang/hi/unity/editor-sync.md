# Editor Sync

Needle, Unity और ब्राउज़र के बीच एक बहुत तेज़, पुनरावृत्त कार्यप्रवाह (iterative workflow) की अनुमति देता है। आमतौर पर, निर्यात में कुछ ही सेकंड लगते हैं। हालांकि, एक बार जब दृश्य अधिक जटिल हो जाते हैं, तो कुछ प्रकार के परिवर्तनों (जैसे मटेरियल प्रॉपर्टीज़ को समायोजित करना, ऑब्जेक्ट्स को थोड़ा हिलाना) के लिए, हम आपके परिवर्तनों को देखने का एक और भी तेज़ तरीका प्रदान करते हैं - **Editor Sync**।

## Editor Sync का उपयोग कैसे करें

आप `EditorSync` कंपोनेंट को अपने दृश्य में जोड़कर Editor Sync सक्षम कर सकते हैं। यह कंपोनेंट आपके Unity Editor को आपके ब्राउज़र प्रोजेक्ट से कनेक्ट करेगा और दोनों के बीच लागू होने वाले परिवर्तनों को स्वचालित रूप से सिंक करेगा।

:::tip
**Editor Sync** वर्तमान में एक प्रायोगिक सुविधा (experimental feature) है। कृपया हमें इसके साथ अपने अनुभव के बारे में बताएं! हम आपकी प्रतिक्रिया जानने के लिए उत्सुक हैं।
:::
